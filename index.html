CA Policy Analyzer

Microsoft Entra ID 条件付きアクセスポリシーをローカルで分析

NIST SP 800-53 DISA STIG Microsoft Baselines

概要

CA Policy Analyzer は、Microsoft Entra ID の条件付きアクセス (CA) ポリシーをローカル環境で分析するセキュリティツールです。ポリシー JSON をインポートするだけで、国際標準に基づくギャップ分析と改善提案を即座に確認できます。

無料機能

ポリシー管理

JSON ファイルの読み込み・一覧表示・検索。変更履歴の自動追跡と差分検出。

ギャップ分析

39 項目のベストプラクティスとの適合チェック。管理者除外の補償判定を含む多角的分析。

セキュリティ評価

総合スコア・重要度別分類・優先対応事項を一画面で確認。

監査ログ

ポリシー変更の履歴管理。前回との差分を自動検出して可視化。

Pro 機能 PRO

  • Excel 設計書エクスポート — 監査提出用 8 シート構成
  • PDF レポート出力 — A4 印刷最適化
  • What-If シミュレーション — 複数シナリオの比較検証
  • ポリシー JSON エクスポート

プライバシー: すべてのデータはローカルで処理されます。外部サーバーへのデータ送信は一切行いません（Microsoft Store ライセンス API を除く）。

システム要件

  • Windows 10 バージョン 2004 (19041) 以降 / Windows 11
  • WebView2 Runtime（通常は Windows に同梱）
  • ディスク: 約 100 MB / RAM: 512 MB 以上

CA Policy Analyzer

Analyze Microsoft Entra ID Conditional Access policies locally

NIST SP 800-53 DISA STIG Microsoft Baselines

Overview

CA Policy Analyzer is a security tool that analyzes Microsoft Entra ID Conditional Access (CA) policies in a local environment. Simply import your policy JSON to instantly review gap analysis and improvement recommendations based on international security standards.

Free Features

Policy Management

Load, list, and search JSON files. Automatic change tracking with diff detection.

Gap Analysis

Compliance check against 39 best practices. Multi-faceted analysis including admin exclusion compensation.

Security Assessment

Overall score, severity classification, and priority actions in a single view.

Audit Log

Policy change history management. Automatic diff detection and visualization.

Pro Features PRO

  • Excel Design Document Export — 8-sheet audit-ready format
  • PDF Report Output — A4 print-optimized
  • What-If Simulation — Multi-scenario comparison
  • Policy JSON Export

Privacy: All data is processed locally. No data is ever sent to external servers (except Microsoft Store license API).

System Requirements

  • Windows 10 version 2004 (19041) or later / Windows 11
  • WebView2 Runtime (typically bundled with Windows)
  • Disk: ~100 MB / RAM: 512 MB or more
