よくある質問

Q. データはどこに保存されていますか？

すべてのデータは以下のフォルダに保存されます:

%LocalAppData%\CAPolicyAnalyzer\

主なファイル:

ファイル 内容
Data/policies.json 登録済みのポリシーデータ
Data/auditlog.json 変更履歴（監査ログ）
Data/named-locations.json ネームドロケーション
Data/analysis-settings.json 分析設定
Data/locale.json 言語設定
eula.dat 利用規約の同意状態

Q. データは外部に送信されますか？

いいえ。すべてのデータはローカルデバイス上でのみ処理・保存されます。

ネットワーク通信は Microsoft Store のライセンス確認のみです。

Q. ポリシー JSON はどのように取得しますか？

Microsoft Entra 管理センターからエクスポートするか、Graph API（GET /identity/conditionalAccess/policies）を使用して取得します。

詳しくは「はじめに」をご覧ください。

Q. 言語を切り替えるにはどうすればいいですか？

About 画面から表示言語を日本語または英語に切り替えられます。

設定は即座に反映されます。

Q. 試用期間が終了したらデータは消えますか？

いいえ。データはそのまま保持されます。

Standard または Pro アドオンを購入すると、すべてのデータを引き続き利用できます。

トラブルシューティング

アプリが起動しない

  1. Windows のバージョンが 10 (19041) 以降であることを確認
  2. WebView2 Runtime がインストールされていることを確認
  3. %LocalAppData%\CAPolicyAnalyzer\ フォルダに書き込み権限があることを確認
  4. 問題が解決しない場合は、診断バンドルを作成してサポートに送信

JSON ファイルを読み込めない

ファイルが正しい JSON 形式であることを確認してください。

Microsoft Entra 管理センターまたは Graph API からエクスポートされた形式に対応しています。

ファイルサイズが 10 MB 未満であることを確認してください。

アプリにはサンプル JSON ファイルが同梱されています。まずサンプルファイルで動作を確認してみてください。

ギャップ分析の結果がおかしい

分析設定で管理者グループ名が正しく設定されているか確認してください。

ポリシーの「状態」が「有効」になっているか確認してください（無効やレポート専用は分析対象外の場合があります）。

エクスポートが失敗する

分析設定で「組織名」と「作成者」が入力されていることを確認してください。

ポリシーが 1 件以上読み込まれていることを確認してください。

データのリセット

アプリのデータを完全にリセットするには:

  1. アプリを終了
  2. エクスプローラーで %LocalAppData%\CAPolicyAnalyzer\ を開く
  3. Data フォルダ内のファイルを削除
  4. アプリを再起動

リセットすると、すべてのポリシーデータ、変更履歴、分析設定が削除されます。必要に応じて事前にバックアップを取ってください。EULA の同意状態もリセットされます。

サポートへの問い合わせ

解決しない問題は、以下の手順でサポートにご連絡ください:

  1. About 画面を開く
  2. 「診断バンドルを作成」をクリック（ポリシーデータは自動的に匿名化されます）
  3. 生成された ZIP ファイルをメールに添付
  4. support@sumikkolab.com に送信

対応言語: 日本語・英語

CA Policy Analyzer User Manual

Frequently Asked Questions

Q. Where is my data stored?

All data is stored in the following folder:

%LocalAppData%\CAPolicyAnalyzer\

Key files:

File Contents
Data/policies.json Registered policy data
Data/auditlog.json Change history (audit log)
Data/named-locations.json Named locations
Data/analysis-settings.json Analysis settings
Data/locale.json Language settings
eula.dat EULA acceptance state

Q. Is my data sent externally?

No. All data is processed and stored only on your local device.

Network communication is limited to Microsoft Store license verification only.

Q. How do I obtain the policy JSON?

Export it from the Microsoft Entra admin center, or use the Graph API (GET /identity/conditionalAccess/policies) to retrieve it.

See "Getting Started" for details.

Q. How do I switch the language?

You can switch the display language between Japanese and English from the About screen.

The setting takes effect immediately.

Q. Will my data be lost when the trial period ends?

No. Your data is retained as-is.

After purchasing the Standard or Pro add-on, you can continue using all your data.

Troubleshooting

The App Won't Start

  1. Verify that your Windows version is 10 (19041) or later
  2. Verify that WebView2 Runtime is installed
  3. Verify that you have write permissions to the %LocalAppData%\CAPolicyAnalyzer\ folder
  4. If the problem persists, create a diagnostic bundle and send it to support

Cannot Load JSON File

Make sure the file is in valid JSON format.

The app supports formats exported from the Microsoft Entra admin center or Graph API.

Verify that the file size is under 10 MB.

The app includes sample JSON files. Try loading a sample file first to verify that the app is working correctly.

Gap Analysis Results Seem Incorrect

Check that administrator group names are correctly configured in the analysis settings.

Verify that the policy "State" is set to "Enabled" (disabled or report-only policies may be excluded from analysis).

Export Fails

Verify that "Organization Name" and "Author" are entered in the analysis settings.

Verify that at least one policy has been loaded.

Data Reset

To completely reset the app data:

  1. Close the app
  2. Open %LocalAppData%\CAPolicyAnalyzer\ in File Explorer
  3. Delete the files inside the Data folder
  4. Restart the app

Resetting will delete all policy data, change history, and analysis settings. Back up your data beforehand if needed. The EULA acceptance state will also be reset.

Contacting Support

For unresolved issues, please contact support using the following steps:

  1. Open the About screen
  2. Click "Create Diagnostic Bundle" (policy data is automatically anonymized)
  3. Attach the generated ZIP file to an email
  4. Send it to support@sumikkolab.com

Supported languages: Japanese and English
