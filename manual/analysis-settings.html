分析設定とは

左サイドバーの「分析設定」セクションでは、ギャップ分析やカバレッジ分析の精度を向上させるための設定を行います。

主に、管理者グループ名とゲストグループ名を登録します。

管理者グループ名の設定

組織で使用している管理者グループの名前を登録します。

入力欄にグループ名を入力し、Enter キーで追加します。

複数のグループ名を登録できます（例: Global Admins, IT Administrators, Security Admins）。

この設定は以下の分析に影響します:

  • 管理者除外分析: どのポリシーが管理者を除外しているかの判定
  • 管理者保護不備の検出: 管理者専用ポリシーの有無の確認
  • 設計不備の検出: 管理者が保護されていないケースの特定

グループ名は完全一致で判定されます。ポリシー JSON に含まれるグループ名と正確に一致させてください。

ゲストユーザーグループ名の設定

ゲストユーザーが所属するグループの名前を登録します。

入力方法は管理者グループ名と同じです。

この設定は以下の分析に影響します:

  • ゲストユーザー保護の評価: ゲスト向けのポリシーが適切に設定されているかの確認

エクスポートメタデータ

Excel / PDF エクスポート時に使用される組織情報を設定します。

  • 組織名 / テナント名（必須）
  • テナント ID（推奨）
  • ドキュメントバージョン（デフォルト: 1.0）
  • 作成者（必須）

設定の保存

分析設定は自動的に保存されます。

保存先: %LocalAppData%\CAPolicyAnalyzer\Data\analysis-settings.json

分析設定を変更した後は、ギャップ分析を再実行すると新しい設定が反映されます。

What Are Analysis Settings?

The "Analysis Settings" section in the left sidebar allows you to configure settings that improve the accuracy of gap analysis and coverage analysis.

Primarily, you register administrator group names and guest group names.

Administrator Group Name Settings

Register the names of administrator groups used in your organization.

Type a group name in the input field and press Enter to add it.

Multiple group names can be registered (e.g., Global Admins, IT Administrators, Security Admins).

This setting affects the following analyses:

  • Administrator Exclusion Analysis: Determines which policies exclude administrators
  • Administrator Protection Gap Detection: Checks whether dedicated administrator policies exist
  • Design Flaw Detection: Identifies cases where administrators are not protected

Group names are matched exactly. Make sure they precisely match the group names contained in the policy JSON.

Guest User Group Name Settings

Register the names of groups that guest users belong to.

The input method is the same as for administrator group names.

This setting affects the following analyses:

  • Guest User Protection Evaluation: Checks whether policies for guest users are properly configured

Export Metadata

Configure organization information used when exporting to Excel or PDF.

  • Organization Name / Tenant Name (required)
  • Tenant ID (recommended)
  • Document Version (default: 1.0)
  • Author (required)

Saving Settings

Analysis settings are saved automatically.

Save location: %LocalAppData%\CAPolicyAnalyzer\Data\analysis-settings.json

After changing analysis settings, re-run the gap analysis to apply the new settings.
