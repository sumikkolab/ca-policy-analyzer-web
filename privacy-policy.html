プライバシーポリシー

CA Policy Analyzer — SumikkoLab｜最終更新日: 2026-03-01

1. データの収集

CA Policy Analyzer は、ユーザーのポリシーデータを一切収集しません。すべてのデータはローカルデバイス上でのみ処理・保存されます。

2. 外部通信

本アプリは以下の場合にのみネットワーク通信を行います:

  • Microsoft Store API — ライセンス確認および購入処理

これ以外の外部サーバーへのデータ送信は一切行いません。ポリシーの分析処理はすべてローカルで実行されます。

3. ローカルデータ

以下のデータがローカルに保存されます:

  • ポリシー JSON ファイル (%LocalAppData%\CAPolicyAnalyzer\Data\)
  • 分析設定（管理者グループ・ゲストグループの名前）
  • 変更履歴（監査ログ）
  • EULA 同意状態
  • 表示言語設定
  • 試用状態（非 Store 環境のみ）

4. 第三者への提供

ユーザーデータを第三者に提供・販売・共有することはありません。

5. サポート時のデータ取り扱い

不具合調査のために提供されたデータは、調査目的以外に使用せず、調査完了後速やかに（原則 30 日以内）破棄します。機密情報を含むデータをそのまま送付しないでください。必ずアプリ内の匿名化機能（診断バンドル作成）をご利用ください。

6. お子様のプライバシー

本アプリは IT 管理者向けの業務ツールであり、13 歳未満のお子様を対象としていません。お子様から意図的に個人情報を収集することはありません。

7. ポリシーの変更

本プライバシーポリシーを変更する場合は、本ページを更新し、最終更新日を改定します。

8. お問い合わせ

プライバシーに関するお問い合わせは、以下のメールアドレスまでご連絡ください。

support@sumikkolab.com

Privacy Policy

CA Policy Analyzer — SumikkoLab | Last updated: March 1, 2026

1. Data Collection

CA Policy Analyzer does not collect any user policy data. All data is processed and stored exclusively on the local device.

2. Network Communication

This app only communicates over the network in the following case:

  • Microsoft Store API — License verification and purchase processing

No data is ever sent to any other external servers. All policy analysis is performed locally.

3. Local Data

The following data is stored locally:

  • Policy JSON files (%LocalAppData%\CAPolicyAnalyzer\Data\)
  • Analysis settings (admin group and guest group names)
  • Change history (audit log)
  • EULA consent status
  • Display language preference
  • Trial status (non-Store environments only)

4. Third-Party Sharing

We do not provide, sell, or share user data with any third parties.

5. Data Handling During Support

Data provided for investigation purposes will not be used for any other purpose and will be promptly destroyed (within 30 days) after investigation is complete. Please do not send data containing confidential information directly. Always use the in-app anonymization feature (diagnostic bundle creation).

6. Children's Privacy

This app is a business tool designed for IT administrators and is not intended for children under 13. We do not knowingly collect personal information from children.

7. Changes to This Policy

If we make changes to this privacy policy, we will update this page and revise the last updated date.

8. Contact

For privacy-related inquiries, please contact us at:
