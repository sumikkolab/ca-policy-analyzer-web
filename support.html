不具合の報告

不具合を報告する際は、以下の手順で診断バンドルを作成してお送りください。診断バンドルにはポリシーデータが匿名化された状態で含まれるため、安全にお送りいただけます。

  1. About 画面を開き、バージョン番号を 5 回クリックして開発者ツールを有効化
  2. DevTools タブで「診断バンドルを作成」をクリック
  3. 生成された ZIP ファイルをメールに添付して送信

機密情報について: 診断バンドルに含まれるポリシーデータはすべて匿名化されています（グループ名・ユーザー ID 等は RedactId に変換済み）。元のデータをそのまま送付しないでください。

よくある質問 (FAQ)

Q. データは外部に送信されますか？

いいえ。すべてのポリシー分析はローカルで実行されます。ネットワーク通信は Microsoft Store のライセンス確認のみです。

Q. 試用期間が終了しました。どうすればよいですか？

Microsoft Store からアプリを購入してください。購入後、すべての基本機能が引き続き利用できます。Pro アドオンを追加すると、Excel/PDF エクスポートやシミュレーション機能も利用できます。

Q. ポリシー JSON はどこから取得できますか？

Microsoft Entra 管理センター、または PowerShell（Microsoft Graph API）を使用してエクスポートできます。詳しくはアプリ内の説明をご覧ください。

Q. 対応している OS は？

Windows 10 バージョン 2004 (19041) 以降、および Windows 11 に対応しています。WebView2 Runtime が必要です（通常は Windows に同梱）。

Q. 組織で複数ライセンスを購入できますか？

Microsoft Store のボリュームライセンス（ビジネス向け Microsoft Store）をご利用いただけます。詳しくはお問い合わせください。

Q. セキュリティ監査のためにソースコードを確認できますか？

法人利用において、セキュリティ監査等の理由でソースコードの確認が必要な場合は、秘密保持契約（NDA）の下で開示を検討可能です。お問い合わせください。

Reporting Issues

When reporting a bug, please create a diagnostic bundle using the following steps. The diagnostic bundle contains anonymized policy data, so it is safe to share.

  1. Open the About screen and click the version number 5 times to enable Developer Tools
  2. In the DevTools tab, click "Create Diagnostic Bundle"
  3. Attach the generated ZIP file to your email

Regarding confidential data: All policy data in the diagnostic bundle is anonymized (group names, user IDs, etc. are converted to RedactId). Please do not send raw data directly.

Frequently Asked Questions (FAQ)

Q. Is any data sent externally?

No. All policy analysis is performed locally. Network communication is limited to Microsoft Store license verification only.

Q. My trial period has expired. What should I do?

Please purchase the app from the Microsoft Store. After purchase, all basic features remain available. Adding the Pro add-on enables Excel/PDF export and simulation features.

Q. Where can I get the policy JSON?

You can export it from the Microsoft Entra admin center or via PowerShell (Microsoft Graph API). See the in-app instructions for details.

Q. What operating systems are supported?

Windows 10 version 2004 (19041) or later, and Windows 11. WebView2 Runtime is required (typically bundled with Windows).

Q. Can I purchase multiple licenses for my organization?

You can use Microsoft Store volume licensing (Microsoft Store for Business). Please contact us for details.

Q. Can I review the source code for a security audit?

For enterprise use, source code review for security audit purposes can be arranged under a non-disclosure agreement (NDA). Please contact us.
