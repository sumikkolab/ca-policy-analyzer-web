不具合の報告

不具合を報告する際は、以下の手順で診断バンドルを作成してお送りください。診断バンドルにはポリシーデータが匿名化された状態で含まれるため、安全にお送りいただけます。

  1. About 画面を開く
  2. 「診断バンドルを作成」をクリック
  3. 生成された ZIP ファイルをメールに添付して送信

機密情報について: 診断バンドルに含まれるポリシーデータはすべて匿名化されています（グループ名・ユーザー ID 等は自動的に変換済み）。元のデータをそのまま送付しないでください。

Reporting Issues

When reporting a bug, please create a diagnostic bundle using the following steps. The diagnostic bundle contains anonymized policy data, so it is safe to share.

  1. Open the About screen
  2. Click "Create Diagnostic Bundle"
  3. Attach the generated ZIP file to your email

Regarding confidential data: All policy data in the diagnostic bundle is anonymized (group names, user IDs, etc. are automatically converted). Please do not send raw data directly.
